# What is Loon? 

**Loon** is a powerful visualization system designed to help you explore, understand, and debug complex biological data. It achieves this by seamlessly integrating three fundamental data types: cell images, time-series data, and tree structures.

At its core, Loon employs a principle of "primary data type driven layout." This means you choose one data type (e.g., cell images) to dictate the overall organization of your visualization, and Loon intelligently weaves the other related data types (time-series, tree data) directly into that layout.
