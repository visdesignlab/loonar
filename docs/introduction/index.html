What is Loon

A Data Visualization Software for Live Cell Microscopy​

Loon is designed to help scientists

  • Analyze their cell data
  • Identify data processing mistakes
  • Communicate their findings
Note: Loon is strictly a visualization tool. Your cell data must be pre-generated using other software (e.g. LiveCyte).

Loon does not:

  • Produce cell segmentation outlines itself
  • Derive metadata (e.g. mass, time, segmentations) from images.

Loon is designed to visualize your cell images and metadata as input.
Loon does not alter your original microscopy data.

Key Features​

Visualize Images, Time-Series Data, and Cell Lineages Together​

Analyze Images with Linked Cell Metadata​

Handle Large-Scale and Multimodal Data by Finding Exemplar Cells​

Observe Cell Metadata Evolve Over Time​

Explore Differences Between Experimental Conditions​
